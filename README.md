## Installation & Setup

1. Clone the repository:

   ```bash
   git clone https://github.com/xZegAs/e-commerce-app-fresh-cart.git
   cd e-commerce-app-fresh-cart
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Run the development server:

   ```bash
   npm start
   ```

4. Open the app in your browser:
   ```
   http://localhost:5173
   ```
